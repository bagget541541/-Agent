## 概览
本文档提供微信公众号文章类别的识别特征和关键信息提取要点，帮助准确分类并提取核心内容。

### 通用要求：文章标题
每篇文章都需要提取或生成一个简洁明了的标题（10-15字），用于标识文章内容。

**标题来源优先级**：
1. 优先提取原文标题（文章开头的标题）
2. 如果原文标题过长或不明确，基于核心内容生成简洁标题
3. 标题格式建议：`[核心词] + [关键信息]`
   - 信用卡活动：`[银行] + [活动类型]`，如"招商银行消费返现活动"
   - 权益变更：`[变更内容]`，如"机场贵宾厅服务调整"
   - 新卡：`[卡种名称]`，如"星座信用卡首发上市"
   - 公告：`[公告类型]`，如"系统维护通知"

## 信用卡活动

### 识别特征
- 标题关键词：活动、优惠、返现、满减、积分、抽奖、福利
- 内容特征：有时间范围、优惠规则、参与条件、名额限制
- 常见表述："限时活动"、"专属福利"、"消费满额"

### 提取要点
- **活动内容**：核心优惠是什么（返现比例、礼品、积分倍数等）
- **活动时间**：具体日期范围（格式：YYYY.MM.DD-YYYY.MM.DD）
- **适用人群**：哪些用户可参与（卡等级、地区、新老用户等）

### 示例
```
活动内容：消费满500元返50元京东卡
活动时间：2025.01.15-2025.02.15
适用人群：招商银行金卡及以上用户
```

## 权益变更

### 识别特征
- 标题关键词：调整、变更、升级、取消、停用、权益更新
- 内容特征：新旧权益对比、生效日期、受影响卡种
- 常见表述："权益调整通知"、"服务变更说明"

### 提取要点
- **消息时间**：公告发布时间（格式：YYYY.MM.DD）
- **影响范围**：哪些卡种或用户群体受影响
- **变更内容**：具体变更了什么权益（增/删/改）
- **变更分析**：变更的影响和趋势（权益增强/缩减/优化）

### 示例
```
消息时间：2025.01.10
影响范围：白金卡用户
变更内容：机场贵宾厅服务从每年6次调整为4次
变更分析：权益缩减，可能影响高端用户体验
```

## 公告或其他

### 识别特征
- 标题关键词：公告、通知、声明、重要提示
- 内容特征：系统升级、服务暂停、风险提示、合规说明
- 常见表述："系统维护公告"、"风险提示"

### 提取要点
- **消息内容**：核心通知信息摘要（维护时间、服务调整、风险提示等）
- **点评**：对公告的简要分析或影响评估（重要程度、影响范围、建议措施等）

### 示例
```
消息内容：系统将于2025.01.20凌晨进行升级维护，届时部分服务暂停
点评：常规维护，影响较小，建议提前完成业务办理
```

## 新发行信用卡

### 识别特征
- 标题关键词：新卡、首发、上市、发行、推出、发布
- 内容特征：卡种介绍、权益亮点、申请条件、年费说明
- 常见表述："全新上市"、"首发权益"、"专属卡面"

### 提取要点
- **卡种**：信用卡完整名称和等级
- **卡亮点**：核心权益和特色（免年费条件、专属权益等）
- **适用人群**：目标用户特征（年龄段、职业、消费习惯）
- **来源**：发卡银行
- **详情**：申请条件、年费政策、其他注意事项

### 示例
```
卡种：招商银行星座信用卡·射手座版
卡亮点：消费1元积1分，首年免年费，生日双倍积分
适用人群：年轻消费群体，星座文化爱好者
来源：招商银行
详情：年刷满6次免次年年费，支持移动支付
```

## 通用提取原则

1. **精简表达**：去除冗余修饰，保留核心信息
2. **格式统一**：严格按照各类别字段要求输出
3. **直接提取**：从文章原文提取，不添加外部信息
4. **准确分类**：优先识别主类别，避免混淆
5. **时间格式**：统一使用 YYYY.MM.DD 格式
