# 微信公众号 biz 参数获取教程

## 什么是 biz？

`biz` 是微信公众号的唯一标识符，用于在 RSSHub 中订阅公众号文章。格式类似：`MzI3NjA3MTYxMA==`

## 获取方法

### 方法一：手机端获取（推荐）

1. 在微信中打开目标公众号的任意一篇文章
2. 点击右上角 `...` → 复制链接
3. 将链接发送到电脑（文件传输助手）
4. 在浏览器中打开该链接
5. 查看地址栏，找到 `__biz=` 参数

```
https://mp.weixin.qq.com/s/xxxxx?__biz=MzI3NjA3MTYxMA==&mid=2247483647&idx=1&sn=xxxxx
                                    ^^^^^^^^^^^^^^^^^^^^^
                                    这就是 biz 参数
```

### 方法二：电脑端获取

1. 在电脑微信中打开公众号文章
2. 右键 → 复制链接
3. 在浏览器中打开
4. 同样在地址栏找到 `__biz=` 参数

### 方法三：通过 RSSHub 自动发现

访问以下地址，输入公众号名称搜索：

```
https://rsshub.app/wechat/mp/search/公众号名称
```

## 配置示例

在 `data/wechat_monitor/config.json` 中添加：

```json
{
  "accounts": [
    {
      "name": "秦农银行信用卡",
      "biz": "从上面步骤获取的 biz 值",
      "category": "银行信用卡",
      "enabled": true
    }
  ]
}
```

## 验证 biz 是否正确

运行以下命令测试：

```bash
cd wechat-rss-monitor
python scripts/rsshub_fetcher.py --biz YOUR_BIZ_VALUE
```

如果返回文章列表，说明 biz 正确。

## 常见问题

### Q: biz 值会变吗？

A: 一般情况下不会变，但公众号迁移或更名时可能会变。

### Q: 获取不到文章怎么办？

A: 检查以下几点：
1. biz 值是否正确（注意大小写和特殊字符）
2. RSSHub 公共实例是否可用
3. 公众号是否设置了隐私保护

### Q: 如何添加更多公众号？

A: 按照上面的步骤获取 biz 值，然后添加到 config.json 的 accounts 数组中即可。
